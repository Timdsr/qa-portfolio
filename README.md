# qa-portfolio

This repository contains a QA junior portfolio based on realistic testing scenarios.

## Skills
- Manual testing
- Test case design
- Bug reporting
- REST API testing

## Tools
- Jira Software
- Google Sheets
- Postman
- Xray (basic)

## Content
- **test-cases/**: Manual test cases created from user stories
- **bug-reports/**: Bug reports written in Jira style
- **api-testing/**: API testing examples using Postman

## Context
All examples are based on a web application authentication feature (login).

This portfolio reflects how I would work as a QA in a real company environment.
